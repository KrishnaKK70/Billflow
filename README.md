# BillFlow

BillFlow is a smart invoice and planner management application designed to simplify customer billing, product planning, and invoice generation.

## Features

* Customer management
* Planner management
* Multiple product assignment
* Invoice generation
* Dynamic product loading
* Billing dashboard
* Product-wise invoice calculation
* Responsive UI

## Technologies Used

* React
* TypeScript
* Firebase / Supabase
* Tailwind CSS
* Node.js

## Functionalities

* Assign multiple products to customers
* Automatically load planner products into invoices
* Generate invoices dynamically
* Manage customer billing workflows
* Product dropdown management

## Future Enhancements

* PDF invoice export
* GST calculation
* Email invoice sharing
* Dashboard analytics
* Role-based access
